Integración con Facturapi

ServiGest usa Facturapi para timbrar facturas CFDI ante el SAT. Necesitas una cuenta en facturapi.io y tu API key.

💡 Modo de prueba: Usa tu API key de sandbox para probar sin costo. Para facturar de verdad usa tu key de producción.
Encriptada y guardada de forma segura. Nunca la compartimos.

CÓMO OBTENER TU API KEY

  1. Crea una cuenta en facturapi.io
  2. Ve a Configuración → Llaves de API
  3. Copia tu Secret Key (empieza con sk_)
  4. Pégala en el campo de arriba y guarda
  5. Ya puedes timbrar facturas CFDI desde ServiGest
